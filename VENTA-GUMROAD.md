## 4. Descripción completa (cuerpo del listing — Markdown)

### ⚡ Claude Code, pero con superpoderes

¿Usas **Claude Code** y sientes que se te va el presupuesto en tokens, que "olvida" tu proyecto cuando es grande, o que repites las mismas tareas una y otra vez?

Este kit lo arregla. **Un solo comando** y Claude Code gana 10 habilidades nuevas que trabajan solas.

```bash
git clone https://github.com/juanmanuel767/protocolo-eficiencia.git
cd protocolo-eficiencia && ./install.sh
```

Eso es todo. Idempotente, reversible (`./uninstall.sh`) y con un **manual de bienvenida** que aparece solo la primera vez.

---

### 💰 Ahorra dinero en cada sesión
- **Subagentes baratos** (`explorador` con Haiku, `revisor` con Sonnet) leen tu código en contexto aislado y solo devuelven conclusiones. Esas lecturas **no inflan** tu conversación principal.
- **Reglas de eficiencia** automáticas: lecturas quirúrgicas, sin narración inútil, contexto fijo delgado.
- `/ahorro`: audita el gasto de la sesión y te dice el ajuste de mayor impacto.

### 🧠 Recuerda proyectos enteros (por grandes que sean)
- `/memoria`: memoria persistente del proyecto (mapa del código + decisiones + pendientes) que **escala a cualquier tamaño** porque guarda resúmenes, no volcados.
- Se **carga sola** en cada sesión y viaja con el repo (se versiona en git).

### 📱 Claude ve y controla tu teléfono
- `/movil`: Claude **captura tu pantalla Android** y la **maneja** (toca, escribe, desliza, abre apps) por cable USB. Funciona en **cualquier** teléfono.
- Automatiza tareas, prueba apps que construyes, manos libres.

### 🔒 Blinda tu código antes de publicarlo
- `/seguridad`: escáner defensivo de **secretos filtrados, archivos sensibles en git y dependencias vulnerables**. Evita el error caro de subir una API key.

### 🚀 Acelera tu día a día
- `/commit`: mensajes de commit convencionales perfectos a partir de tu diff.
- `/pruebas`: detecta el framework, corre los tests y genera los que faltan.
- `/voz`: EMMA te responde **hablando** (estilo JARVIS).
- `/super` y `/ultra`: modo máxima capacidad e ingeniería avanzada.

## 7. Preguntas frecuentes (FAQ)

**¿Necesito saber configurar Claude Code?**
No. Un comando lo instala y configura todo.

**¿Funciona en Windows?**
El núcleo sí (skills y reglas). Los helpers de shell y `/movil` están pensados para Linux/macOS; en Windows usa WSL.

**¿`/movil` funciona en iPhone?**
Está optimizado para Android por USB. iOS requiere Mac + Xcode (limitado).

**¿Es seguro? ¿Claude va a tocar cosas peligrosas?**
El kit **bloquea a propósito** los comandos destructivos (sudo, rm, push forzado, credenciales). `/seguridad` es solo defensa de tu propio proyecto.

**¿Puedo usarlo en proyectos comerciales?**
Sí, licencia MIT.

**¿Recibo actualizaciones?**
Sí, el repo se actualiza y tu compra te da acceso.
